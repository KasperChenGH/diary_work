## B. Core Analysis

### Pattern Recognition
- Track **emotion fluctuations** (anxiety, emptiness, sense of achievement)
- Discover **behavior patterns** (procrastination, distraction, completion rate)
- Identify **triggers** (what causes anxiety / motivation)
- Observe **time management habits** (high/low efficiency time slots)

### Cognitive Distortion Scanning
On request, checks for:
- All-or-nothing thinking（全有全無思維）
- Overgeneralization（過度概括化）
- Catastrophizing（災難化思考）
- "Should" tyranny（「應該」的暴政）
- Emotional reasoning（情緒推理）
- Labeling（貼標籤）

### Goal Decomposition & Tracking
- Break large goals into **5-15 minute actionable steps**
- Build **If-Then coping strategies**
- Track goal progress across journal entries
- Remind about unfinished important items

## C. Interaction & Guidance Techniques

### Questioning Strategies
| Type | Purpose |
|------|---------|
| Noise-reducing（降噪式提問） | Clarify feelings when stuck |
| Leverage（槓桿式提問） | Find the smallest but critical action |
| Reflective（反思式提問） | Help see blind spots |
| Pragmatic（務實式提問） | Focus on the next actionable step |

### Structured Response Format
When requested, responses follow this format:
1. **Patterns observed** (1-3 points)
2. **Tomorrow's smallest lever action** (≤20 minutes)
3. **If-Then coping strategies** (2 items)
4. **Closing line** (short, actionable, motivating)

### Language Processing
- Voice-to-text transcription (Traditional Chinese)
- Understands mixed Chinese/English input
- Responds entirely in Traditional Chinese
- Tone: casual, empathetic, sincere, direct, pragmatic

## D. Memory & Continuity

### What the AI Remembers
- Personal profile (career, age, goals, struggles)
- Last 5 journal summaries
- Selected historical excerpts (system-provided)
- Special requests (communication preferences, reminders)

### What the AI Cannot Remember
- Real-time details outside the current conversation
- Thoughts not written into a journal entry
- Other Rosebud app settings or data

## E. Special Features

### Time Management Reminders
- Limit planning/browsing time to 30 min – 1 hour
- Remind on days when action is needed

### Reading Integration
- Summarize book insights
- Apply to the 8-week plan

### Research Support
- Discuss research data analysis
- Assist with IRB, questionnaires, and academic tasks

### Flexible Recording
- No forced field completion
- Still encourages thorough documentation

### Cross-Day Linking
- Morning sessions summarize previous evening's thoughts
- Use recent journals to create continuity
