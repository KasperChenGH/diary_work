# Journal Log

Rolling log of recent journal entries. Claude reads this at the start of each journal session to maintain continuity, recognize patterns, and connect themes across days.

**Rules:**
- Keep only the **last 10 entries** (delete oldest when adding new)
- Each entry is a compact summary, NOT the full journal conversation
- Claude auto-generates each entry at the end of a journal session
